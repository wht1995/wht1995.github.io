# 水平 & 垂直居中

Tips：对于子元素设置了position，父元素也要设置，否则就是相对body 进行定位。

# 块级元素水平居中

  • 已知子元素宽高，对 子元素 使用：margin: 0 auto;
.child{
    width: 200px;
    height: 200px;
	margin: 0 auto;
}
  • 已知子元素宽高，对 父元素 使用：flex 布局
.parent {
        width: 800px;
        height: 600px;
        background-color: #999;
        display: flex;
        justify-content: center;
    }

    .child {
        width: 200px;
        height: 200px;
        background-color: antiquewhite;
    }
  • 已知或未知 子元素宽高，对 子元素 使用：transform + 绝对定位（translate是按自身宽高百分比来平移的）
.child{
    /*width: 200px;
    height: 200px;*/
    background-color: antiquewhite;
    position: absolute;
    left:50%;	//向右偏移了一半，此时元素的左边框已居中，我们需要的是元素的中心点居中
    transform: translate(-50%, 0) //此时再让元素往按自身宽度width的百分比负向移动50%，即可中心居中
}
  • 已知子元素宽高，对 子元素 使用绝对定位 + 负margin
.child {
        width: 200px; 
        height: 200px; 
        background-color: antiquewhite;
        position: absolute;
        left: 50%;
        margin-left: -100px; //这里必须是 width 的 1/2，道理和上面的类似
    }
  • 已知子元素宽高，对 子元素 使用绝对定位 + left/right : 0 + margin:0 auto
.child {
        width: 200px;
        height: 200px;
        position: absolute;
        right: 0;
        left: 0;
        margin: 0 auto;
        background-color: antiquewhite;
    }

# 块级元素垂直居中

  • 已知或未知 子元素宽高，对 父元素 使用：flex + align-items
.parent {
        width: 800px;
        height: 600px;
        background-color: #999;
        display: flex;
        align-items: center;
    }

    .child {
        /*width: 200px;
        height: 200px;*/
        background-color: antiquewhite;
    }
  • 已知子元素宽高， 对 子元素 使用 绝对定位+ 负margin （道理和水平居中差不多）
    .child {
        width: 200px;
        height: 200px;
        background-color: antiquewhite;
        position: absolute;
        top:50%;
        margin-top: -100px;
    }
  • 已知或未知 子元素宽高，对 子元素 使用：transform + 绝对定位（道理和水平居中差不多）
.child {
        /* width: 200px;
        height: 200px; */
        background-color: antiquewhite;
        position: absolute;
        top:50%;
        transform: translate(0,-50%);
    }
  • 已知子元素宽高，对 子元素 使用绝对定位 + top/bottom : 0 + margin:auto 0
.child {
        width: 200px;
        height: 200px;
        position: absolute;
        top: 0;
        bottom: 0;
        margin: auto 0;
        background-color: antiquewhite;
    }

# 行内元素水平居中

对 父元素 设置 text-align:center ，即可实现行内元素水平居中.

# 行内元素垂直居中

.parent {
    height: 高度;
}
.child {
    line-height: 高度;
}
